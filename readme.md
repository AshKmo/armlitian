# Armlitian
This is a compiler for a programming language called Armlitian that compiles to Assembly code for [Peter Higginson's ARMlite simulator](https://peterhigginson.co.uk/ARMlite/).

Run this program with a script file to dump the compiled application to the console, e.g. `./armlitian conway-life.txt`
